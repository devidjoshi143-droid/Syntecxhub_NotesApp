# Notes App

A simple Notes App developed using Kotlin and SQLite in Android Studio.

## Features

- Add Notes
- View Notes
- Edit Notes
- Delete Notes
- Search Notes
- SQLite Database Storage

## Technologies Used

- Kotlin
- SQLite
- RecyclerView
- CardView
- Android Studio

## Database

The application uses SQLiteOpenHelper for local database storage.
